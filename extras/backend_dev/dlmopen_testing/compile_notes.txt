Build shared library:

     g++ -shared -Wl,-soname,libfirst.so libfirst.cpp -o libfirst.so -fPIC

NOTE: If you recompile the library, all the symbol names will change (compiler dependent).
      You will have to update the symbol names in any backend header file you use, e.g. 
      backend_libfirst.hpp. (This will be fixed later...)

      The correct symbol names can be found using the 'nm' command:  nm libfirst.so


Build test program for backends:

     g++ use_backends.cpp -I./ -o use_backends.out -ldl
