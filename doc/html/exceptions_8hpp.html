user method for throwing exceptions deriving from gambit_exception_base

Example use: 

 using namespace gambit::exceptions;
 GAMBIT_THROW_EXCEPTION(e_unphysical << i_text("mass is negative"));

The macro enforces use of gambit_exception_base or derived exceptions, and adds various debug information to e before throwing.
