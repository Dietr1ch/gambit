Detailed Description

Empty example model.

mssmX class. Option nu_dimensions for mssm X may be 24,25, 100, etc..

it's a most trivial example - fine for example when using darksusy who knows how to deal with cmssm, pmssm, mssm24, etc.

Could provide more in other cases, for example doing stuff based on a Lagrangian.
