Detailed Description

Model Parameters Base class example.

Example and base class for typical model parameters. Not so important how set/get works but the model you want to use must be able to understand it

Author:
Johan Lundberg
Date:
July - August 2011

initially based on 'Model' and 'Parameterset' code from Joachim Ripken.
